
# 📊 KPI Definitions

| KPI | Description |
|------|-------------|
| Revenue | Total confirmed booking revenue |
| ADR | Average Daily Rate (Revenue ÷ Occupied Rooms) |
| Occupancy | Occupied Rooms ÷ Available Rooms |
| RevPAR | Revenue per Available Room |
| Booking Count | Total confirmed reservations |
| Room Nights | Total occupied room nights |
| Lead Conversion | Percentage of leads converted into bookings |
| OTA Revenue | Revenue generated through OTA channels |
| Target Achievement | Actual Revenue compared with Target Revenue |
| Average Booking Value | Revenue divided by total bookings |
