
# 📊 Live Dashboard

## Overview

This dashboard provides an interactive view of hotel sales and reservation performance.

It is designed for hotel management to monitor revenue, occupancy, booking trends, lead generation, and OTA performance across multiple properties.

---

## Dashboard Features

- 📈 Revenue Overview
- 🏨 Booking Analysis
- 📊 Sales Trend Analysis
- 🎯 Target vs Achievement
- 👥 Lead Management
- 🌐 OTA Performance

---
