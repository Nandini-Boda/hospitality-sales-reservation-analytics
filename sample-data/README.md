# 📁 Sample Dataset

## Overview

This folder contains anonymized sample datasets used for demonstrating the Hospitality Sales & Reservation Analytics dashboard.

All sensitive business information has been removed or modified.

---

## Files

| File | Description |
|------|-------------|
| sales_sample.csv | Daily sales summary |
| reservations_sample.csv | Sample reservation transactions |
| targets_sample.csv | Monthly revenue targets |

---

## Notes

- Guest information has been removed.
- Property names are for demonstration purposes.
- Revenue figures have been modified while maintaining realistic business patterns.
